MediZen+ 💊

MediZen+ is a Digital Medicine Reminder System developed using HTML, CSS, JavaScript, Spring Boot, and MySQL.

Features
* User Registration & Login
* Medicine Reminders with Alarm
* Medicine Status Tracking (Taken / Missed / Pending)
* Emergency Contact Management
* Doctor Visit History
* Patient Medical History
* Dashboard Analytics
* Responsive UI

Tech Stack

Frontend
* HTML
* CSS
* JavaScript

Backend
* Spring Boot
* REST API

Database
* MySQL
